# Cost Optimizer for AI Solutions

This project provides a cost optimizer for AI solutions. It loads a list of AI solutions from a JSON file, compares their costs and qualities, and recommends the most suitable solution based on a given budget and minimum quality.

## Usage

1. Install the project using `poetry install`.
2. Run the project using `poetry run python src/cost_optimizer.py`.
3. The project will load a list of AI solutions from a JSON file, compare their costs and qualities, and recommend the most suitable solution.

## Testing

1. Install the project using `poetry install`.
2. Run the tests using `poetry run pytest tests`.
