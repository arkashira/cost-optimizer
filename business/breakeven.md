# breakeven.md

## Unit Economics & Break-Even Analysis

### Cost per Active User
- **Compute Costs**: $0.10 per user per month
- **Storage Costs**: $0.05 per user per month
- **Bandwidth Costs**: $0.03 per user per month

**Total Cost per Active User**:  
**$0.10 + $0.05 + $0.03 = $0.18 per user per month**

### Pricing Tiers
| Tier Name         | Price ($/mo) | Features                                           |
|-------------------|--------------|----------------------------------------------------|
| **Basic**         | $9           | Cost optimization insights, Basic API usage report |
| **Pro**           | $29          | All Basic features + Advanced analytics, Alerts    |
| **Enterprise**    | $99          | All Pro features + Custom integrations, Dedicated support |

### Customer Acquisition Cost (CAC) Range
- **CAC Estimate**: $50 - $100 per user

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**: 
  - Basic: $9
  - Pro: $29
  - Enterprise: $99
- **Average Customer Lifespan**: 24 months
- **LTV Calculation**:
  - Basic: $9 * 24 = $216
  - Pro: $29 * 24 = $696
  - Enterprise: $99 * 24 = $2,376

### Break-even Users Count
- **Fixed Costs Estimate**: $5,000 per month
- **Contribution Margin per User**:
  - Basic: $9 - $0.18 = $8.82
  - Pro: $29 - $0.18 = $28.82
  - Enterprise: $99 - $0.18 = $98.82

**Break-even Users Count Calculation**:
- Basic: $5,000 / $8.82 ≈ 567 users
- Pro: $5,000 / $28.82 ≈ 174 users
- Enterprise: $5,000 / $98.82 ≈ 51 users

### Path to $10K MRR
- **Target MRR**: $10,000
- **Monthly Revenue per Tier**:
  - Basic: 567 users * $9 = $5,103
  - Pro: 174 users * $29 = $5,046
  - Enterprise: 51 users * $99 = $5,049

**Combination for $10K MRR**:
- **Pro Tier**: 200 users = $5,800
- **Enterprise Tier**: 25 users = $2,475
- **Basic Tier**: 100 users = $900

**Total Users for $10K MRR**:  
200 (Pro) + 25 (Enterprise) + 100 (Basic) = **325 users**
